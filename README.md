# Project1 Hosting Instructions

This folder contains `Project1.html`, a local event registration form.

## How to publish this page online

1. Create a GitHub account if you do not already have one.
2. Create a new repository, for example `my-first-project`.
3. Upload `Project1.html` to that repository.
4. Enable GitHub Pages in the repository settings.
   - Select the `main` branch or `gh-pages` branch.
   - Save the settings.
5. Your published URL will look like:

   `https://<your-username>.github.io/<repo-name>/Project1.html`

## Example resume link

If your GitHub username is `yourusername` and repo name is `my-first-project`, use:

`https://yourusername.github.io/my-first-project/Project1.html`
